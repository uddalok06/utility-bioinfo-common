### utility-bioinfo-common
various scripts to tackle common immune loci based curations, filtering, file management etc

##### 1. extract_immune_loci_HPRC.sh
  script to extract immune loci from HPRC WSG assemblies
  inputs: 
  i. mat/pat or hap1/hap2 merged.fasta.gz
  ii. sample_name
  hardcoded path inside for specific immune loci flanks

  output:
  sample specific all immune loci fasta
